关于博客

入坑博客还要从 2019 年底，我买了人生中第一台云服务器说起，趁着还没毕业蹭一下学生优惠，114 大洋一年，价格还行。

并不像大多数人一样由大名鼎鼎的 WordPress 入门，我的第一个博客是用 Halo 搭建的，那时候还没听说过静态博客（Hexo 真香），搭建成功后陆陆续续写了几篇，直到 2020 年初，在了解了 Hexo 后，毫不犹豫的决定更换阵营了。

Hexo 的主题更多，可自定义的选项也更多，真的是太对我这个喜欢折腾的人的胃口了。最重要的是，Hexo 可以托管到 GitHub 上，不占用服务器的资源，不担心网络攻击，舒服了……

虽然搭建过程中遇到了大大小小的问题，但总体来讲我还是很满意的，目前还没加入评论系统，等我有空再慢慢施工吧~
